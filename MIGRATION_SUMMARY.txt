================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: MongoDB connection failed
Solution: 
  1. Check if MongoDB is running: Get-Service -Name MongoDB
  2. Start MongoDB: Start-Service -Name MongoDB
  3. Verify MONGODB_URI in backend/config.env

Problem: Dependencies error
Solution:
  1. Delete node_modules folder
  2. Delete package-lock.json
  3. Run: npm install

Problem: Port 5000 in use
Solution:
  1. Change PORT=5001 in backend/config.env
  2. Or kill process: netstat -ano | findstr :5000
